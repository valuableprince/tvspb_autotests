# Автотесты для сайта TVSPB

## Описание
Автоматизированные тесты для проверки функциональности сайта https://tvspb.ru/

## Требования
- Java 11+
- Maven 3.6+
- IntelliJ IDEA

## Установка
1. Клонировать репозиторий
2. Открыть проект в IntelliJ IDEA
3. Дождаться загрузки зависимостей Maven

## Запуск тестов
### Через IntelliJ IDEA
- Правой кнопкой на `testng.xml` → Run 'testng.xml'
- Или: Run → Run 'All Tests'
